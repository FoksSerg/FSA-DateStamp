СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Windows 10/11 (64-bit)
- 100 МБ свободного места на диске
- Права администратора (для установки)

ДОСТУПНЫЕ ВЕРСИИ:

1. FSA-DateStamp-Setup.exe (65.5 МБ)
   - Полный инсталлятор для Windows
   - Устанавливает в Program Files
   - Создает ярлыки на рабочем столе и в меню Пуск
   - Требует права администратора

2. FSA-DateStamp-Portable.zip (63.2 МБ)
   - Портативная версия
   - Не требует установки
   - Можно запускать с USB-флешки
   - Сохраняет настройки в папке с программой

УСТАНОВКА:

Вариант 1 - Полная установка:
1. Запустите FSA-DateStamp-Setup.exe от имени администратора
2. Следуйте инструкциям мастера установки
3. Выберите папку установки (по умолчанию: C:\Program Files\AW-Software\FSA-DateStamp)
4. Выберите создание ярлыков
5. Завершите установку

Вариант 2 - Портативная версия:
1. Распакуйте FSA-DateStamp-Portable.zip в любую папку
2. Запустите FSA-DateStamp.exe
3. Настройки сохраняются в файле datestamp_settings.ini

ИСПОЛЬЗОВАНИЕ:

1. Запустите FSA-DateStamp.exe
2. Выберите исходную папку с изображениями
3. Выберите папку для сохранения результатов
4. Настройте параметры штампа:
   - Размер шрифта (10-100px)
   - Позиция (7 вариантов размещения)
   - Отступы от краев
   - Шрифт (системные шрифты)
5. Нажмите "Обработать изображения"

ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- BMP (.bmp)
- TIFF (.tiff, .tif)

ОСОБЕННОСТИ:
- Автоматическое извлечение даты из EXIF, имени файла или времени создания
- Сохранение структуры папок
- Сохранение метаданных файлов
- Предварительный просмотр штампа
- Интерактивный выбор позиции
- Система логов с фильтрацией

УДАЛЕНИЕ:

Полная версия:
- Используйте "Удаление программ" в Панели управления
- Или запустите uninstall.exe из папки установки

Портативная версия:
- Просто удалите папку с программой

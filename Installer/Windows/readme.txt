FSA-DateStamp v1.01.25.249 - Инструкция по установке

Добро пожаловать в FSA-DateStamp!

Версия: 1.01.25.249 (6 сентября 2025)
Компания: AW-Software
Автор: Сергей Фокин @FoksSerg

ОПИСАНИЕ:
FSA-DateStamp - это система для автоматического добавления водяных знаков с датой и временем на изображения с сохранением структуры папок и метаданных файлов.

ВОЗМОЖНОСТИ:
- Добавление водяных знаков с датой и временем
- Сохранение структуры исходных папок
- Сохранение дат создания и модификации файлов
- Поддержка множества форматов изображений (JPG, PNG, BMP, TIFF)
- Извлечение даты из EXIF, имени файла или времени создания
- 7 позиций размещения штампов
- Настраиваемые отступы от краев изображения
- Графический интерфейс для удобной работы

УСТАНОВКА:
1. Запустите FSA-DateStamp-Setup.exe
2. Следуйте инструкциям мастера установки
3. Выберите папку для установки (по умолчанию: Program Files\FSA-DateStamp)
4. Создайте ярлыки на рабочем столе и в меню Пуск (опционально)
5. Завершите установку

ЗАПУСК:
После установки вы можете запустить FSA-DateStamp:
- Из меню Пуск: Пуск > FSA-DateStamp
- С рабочего стола: двойной клик по ярлыку
- Из папки установки: FSA-DateStamp.exe

ИСПОЛЬЗОВАНИЕ:
1. Выберите папку с исходными изображениями
2. Выберите папку для сохранения результатов
3. Настройте параметры штампа (размер шрифта, позиция, отступы)
4. Нажмите "Обработать" для начала обработки

УДАЛЕНИЕ:
Для удаления программы используйте стандартную процедуру Windows:
Панель управления > Программы и компоненты > FSA-DateStamp > Удалить

ТРЕБОВАНИЯ К СИСТЕМЕ:
- Windows 10 или новее
- 50 МБ свободного места на диске
- Минимум 2 ГБ оперативной памяти

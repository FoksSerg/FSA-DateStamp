Снимок перед созданием версионных сборок для Windows

- Откат к стабильной версии c2d65f1
- Подготовка к созданию двух версий:
  * Windows 7 - Python 3.8-3.9 (отдельная сборка)
  * Windows 8+ - Python 3.13 (универсальная сборка)
- Планируется модификация Build.py для версионных сборок
- Обновление installer.iss для выбора нужной версии

Цель: Максимальная совместимость с минимальными усилиями
